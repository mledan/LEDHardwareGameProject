# LEDHardwareGameProject
A 1-dimensional pong inspired hardware project using the ATTiny85 microcontroller

PARTS LIST
-----------
-ATTiny85 Microcontroller
-USBISP programmer for ATTiny85
-8xNeopixel strip
-wires
